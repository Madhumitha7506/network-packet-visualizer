# Network Packet Flow Visualizer

A web-based visualization tool that simulates how data packets travel across a network.

## Features
- Client → Router → Server simulation
- Packet animation
- Response cycle (Server → Client)
- Status updates

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Covered
- Computer Networks
- Packet Transmission
- Request-Response Model

## How to Run
Open index.html in a browser
